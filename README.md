# TheGioiDiDong Clone - E-Commerce Website

Website thương mại điện tử với giao diện giống **thegioididong.com**, bao gồm đầy đủ chức năng quản lý sản phẩm.

## 🚀 Cách sử dụng

### Mở website
Chỉ cần mở file `index.html` bằng trình duyệt:
```
C:\Users\DELL\Downloads\demo1\index.html
```

### Các trang chính

1. **Trang chủ** (`index.html`)
   - Xem danh sách sản phẩm
   - Tìm kiếm và lọc sản phẩm
   - Flash sale với countdown

2. **Chi tiết sản phẩm** (`product-detail.html`)
   - Click vào bất kỳ sản phẩm nào để xem chi tiết
   - Chọn màu sắc, dung lượng
   - Xem sản phẩm liên quan

3. **Quản lý sản phẩm** (`admin.html`)
   - Click "⚙️ Quản lý" ở header
   - Thêm/sửa/xóa sản phẩm
   - Xem thống kê

## 🎯 Hướng dẫn test

### Test trang chủ
1. Mở `index.html`
2. Xem banner với hiệu ứng pháo hoa
3. Countdown timer đang chạy
4. Hover vào product cards → scale up
5. Click filter tabs để lọc sản phẩm
6. Nhập từ khóa vào search bar

### Test chi tiết sản phẩm
1. Click vào bất kỳ sản phẩm nào
2. Xem thông tin chi tiết
3. Click chọn màu sắc, dung lượng
4. Click "Thêm vào giỏ hàng" → Toast notification

### Test quản lý sản phẩm
1. Click "⚙️ Quản lý" ở header
2. Xem danh sách và thống kê
3. Click "➕ Thêm sản phẩm mới"
4. Điền form và lưu
5. Click ✏️ để sửa sản phẩm
6. Click 🗑️ để xóa sản phẩm
7. Refresh trang → data vẫn còn (localStorage)
